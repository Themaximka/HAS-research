# Публикация опросника в Streamlit Cloud

## 1) Создайте репозиторий на GitHub
1. Создайте новый пустой репозиторий (например `has-survey`).
2. Загрузите в него файлы из папки проекта:
   - `streamlit_app.py`
   - `requirements.txt`
   - `.gitignore`

## 2) Подключите репозиторий к Streamlit Cloud
1. Откройте: https://share.streamlit.io/
2. Нажмите **New app**.
3. Выберите ваш GitHub-репозиторий и ветку.
4. В поле **Main file path** укажите:
   - `streamlit_app.py`
5. Нажмите **Deploy**.

## 3) Получите публичную ссылку
После деплоя появится постоянный URL вида:
`https://<app-name>.streamlit.app`

Эту ссылку можно отправлять респондентам из любой точки.

## Важно про сохранение ответов
Файл `results.csv` в Streamlit Cloud может очищаться при перезапусках контейнера.
Для надежного хранения лучше подключить внешнее хранилище (Google Sheets, Supabase, Postgres и т.д.).
